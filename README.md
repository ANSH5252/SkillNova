## 🧠 What is SkillNova?

SkillNova is an **AI-powered ATS (Applicant Tracking System) simulator** that helps candidates understand how their resumes perform before applying to jobs.

Instead of blindly applying, users can:

- 📄 Upload their resume  
- 🎯 Select a target role  
- 📊 Get realistic ATS scores  
- ⚠️ Identify missing skills  
- ✍️ Generate AI-powered cover letters  

---

## 🌍 Problem

- Millions of resumes get rejected by ATS systems  
- Candidates receive **no feedback**  
- Students don’t know what skills they’re missing  
- Universities lack visibility into skill gaps  

---

## 💡 Solution

SkillNova simulates a **real-world ATS engine**, providing:

- Transparent resume evaluation  
- Skill gap analysis  
- Actionable improvement steps  
- AI-generated cover letters  

---

## ✨ Key Features

### 📄 Resume Analysis (Client-Side)
- PDF parsing happens locally  
- No data upload → **Privacy-first**

---

### 🤖 AI ATS Scoring
- Evaluates:
  - ATS Readability  
  - Technical Match  
- Identifies missing skills  
- Provides system verdict  

---

### ✍️ AI Cover Letter Generator
- Generates role-specific cover letters  
- Highlights strengths  
- Compensates for missing skills  

---

### 📊 Skill Gap Insights
- Shows:
  - Missing technologies  
  - Required improvements

### 🏢 B2B Dashboard (Planned)
- University-level analytics  
- Track student skill gaps  
- Align curriculum with industry

## ⚙️ Local Setup

### 1️⃣ Clone Repository
```bash
git clone https://github.com/Git-Nova-Coders/skillnova.git
cd skillnova
```

### 2️⃣ Install Dependencies
```
npm install
```

### 3️⃣ Add Environment Variables

  Create ```.env.local```:

```
VITE_FIREBASE_API_KEY=your_key
VITE_FIREBASE_AUTH_DOMAIN=your_domain
VITE_FIREBASE_PROJECT_ID=your_project
VITE_FIREBASE_STORAGE_BUCKET=your_bucket
VITE_FIREBASE_MESSAGING_SENDER_ID=your_sender_id
VITE_FIREBASE_APP_ID=your_app_id
VITE_GROQ_API_KEY=your_groq_key
```
### 4️⃣ Run Project
```
npm run dev
```
Open: ```http://localhost:5173```
