# Security Policy

## Supported Versions
At Git Nova Coders, we take the security of SkillNova seriously. Currently, only the latest deployment on the `main` branch receives active security updates.

| Version | Supported          |
| ------- | ------------------ |
| Latest  | :white_check_mark: |
| Older   | :x:                |

## Reporting a Vulnerability

If you discover a security vulnerability within SkillNova, please **DO NOT** create a public GitHub issue. 

Instead, please send a direct message to the repository maintainers or email the core Git Nova Coders team. We will acknowledge receipt of your vulnerability report within 48 hours and strive to send you regular updates about our progress in fixing it.

Please provide:
* A description of the vulnerability.
* The steps required to reproduce the issue.
* Any potential impact on user data or Firebase security rules.
